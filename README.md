# Wizzaro Flexbox Mixins

## Basic information

This is a set of mixins for CSS Flexible Box Layout.
For more information about layout look on: https://www.w3.org/TR/css-flexbox-1/

Browsers Support:
- Chrome 21+
- Firefox 18+
- Opera 12.1+
- Safari 6.1+
- Edge
- IE10+

This mixins old browsers flexbox syntax.
